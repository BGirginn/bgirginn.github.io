LoRa Factory Automation System

LoRa ESP32 Raspberry Pi C++ TUBITAK Industrial IoT

An automated control system developed for Talu Textile as part of a TUBITAK-funded research project. The system uses LoRa wireless technology to enable seamless communication between factory equipment, providing real-time monitoring and control capabilities across large industrial facilities.

Project Overview

  • Started November 2024 as part of TUBITAK grant program
  • Collaboration with Talu Textile for real-world industrial deployment
  • Designed to replace traditional wired control systems with wireless LoRa network
  • Focus on reliability, range, and low power consumption for 24/7 operation

System Architecture

  • Communication: LoRa long-range wireless protocol
  • Nodes: ESP32 microcontrollers for sensor/actuator control
  • Gateway: Raspberry Pi for data aggregation and processing
  • Programming: C++ firmware for ESP32, Python backend for Raspberry Pi
  • Range: Multi-kilometer coverage across factory premises

Technical Implementation

  • Custom LoRa network protocol optimized for industrial environment
  • ESP32 nodes handle local sensor readings and actuator control
  • Raspberry Pi gateway aggregates data from multiple nodes
  • Real-time monitoring dashboard for factory operators
  • Failsafe mechanisms for critical control operations

Applications & Impact

The system enables remote monitoring and control of textile manufacturing equipment, reducing wiring costs and improving flexibility in factory floor layouts. The wireless nature allows for easy reconfiguration and expansion as production needs change.

Bluetooth Arduino Robotics Car

Arduino Bluetooth LIDAR Robotics Competition Winner

A sophisticated robotics project developed for Biga ICDAS Science High School robotics competition. The project simulated a space mining vehicle capable of autonomous navigation, object manipulation, and wireless control - achieving full mission success with highest accuracy among all competitors.

Project Context

  • Developed between April-May 2020 during high school
  • Competition theme: Autonomous space mining vehicle prototype
  • Required integration of multiple sensor systems and robotic manipulation
  • Completed all competition missions with perfect accuracy

Hardware Components

  • Controller: Arduino microcontroller platform
  • Wireless: Bluetooth module for remote control
  • Distance Sensing: Infrared (IR) sensor for proximity detection
  • Navigation: LIDAR sensor for line tracking
  • Manipulation: Electric motor-driven robotic arm
  • Gripper: FSR (Force Sensitive Resistor) pressure sensor for claw control
  • Additional: Ultrasonic sensors for obstacle avoidance

Key Features

  • Precise line following using LIDAR scanning
  • Object detection and distance measurement with IR sensors
  • Motorized robotic arm with multiple degrees of freedom
  • Pressure-sensitive gripper with force feedback
  • Wireless control via Bluetooth for manual override
  • Autonomous navigation between waypoints

Competition Results

The vehicle successfully completed all assigned missions including line tracking, object pickup, transportation, and precise placement. The combination of accurate sensor fusion and reliable mechanical design resulted in perfect mission execution, earning the highest accuracy award in the Science High School robotics competition.
